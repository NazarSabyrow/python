 О проекте
Этот проект представляет собой 2D-игру, разработанную с использованием языка программирования Python и библиотеки PyGame. Она была разработана в рамках Лабораторного практикума №5.
 ВАЖНОЕ ЗАМЕЧАНИЕ
Это консольное приложение PyGame!
- Не работает в GitHub Pages 
- Не работает в веб-браузере 
- Может быть запущено только с Python на вашем компьютере 
Установка и запуск
 Требования
- Python 3.6 или более поздней версии
- Библиотека PyGame
 Пошаговая установка
bash
1. Скачайте проект на свой компьютер
git clone https://github.com/YOUR-USERNAME/python-oyunum.git
2. Войдите в папку проекта
cd python-oyunum
3. Установите необходимую библиотеку
pip install pygame
 4. Запустите игру
python main.py

1. СРЕДА ВЫПОЛНЕНИЯ
JavaScript: Веб-браузер (Chrome, Firefox, Safari)
Python/PyGame: Требует установленного Python и PyGame на компьютере
2. РАСПРОСТРАНЕНИЕ
JavaScript: Веб-сайт - любой может играть по ссылке

 ТЕХНИЧЕСКИЕ РАЗЛИЧИЯ
3. ИГРОВОЙ ЦИКЛ

4. ОБРАБОТКА СОБЫТИЙ
JavaScript: Встроенная система событий браузера
Python/PyGame: Ручной захват событий (pygame.event.get())
5. ГРАФИЧЕСКАЯ СИСТЕМА
JavaScript: Canvas API - пиксельная отрисовка
Python/PyGame: Система Surface и blit - объектно-ориентированная
 РАЗЛИЧИЯ В СТРУКТУРЕ КОДА
6. УПРАВЛЕНИЕ ОБЪЕКТАМИ

7. УПРАВЛЕНИЕ ПАМЯТЬЮ
JavaScript: Автоматическая сборка мусора
Python/PyGame: Требует ручной очистки списков
РАЗЛИЧИЯ В ПОЛЬЗОВАТЕЛЬСКОМ ОПЫТЕ

 ОПЫТ РАЗРАБОТЧИКА
